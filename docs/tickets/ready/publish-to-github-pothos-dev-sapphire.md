## What to build

Publish the project to GitHub as **`pothos-dev/sapphire`**.

- Create the GitHub repository `pothos-dev/sapphire` (public unless told otherwise).
- Add it as the `origin` remote (none is configured today).
- Push the current default branch and set upstream tracking.

This is an **outward-facing, hard-to-reverse** action (a public repo publishes all history). Confirm the repo name, visibility, and that the working tree is in the desired state with the user immediately before pushing. Ensure no secrets or unintended local artifacts are included.

## Acceptance criteria

- [ ] Repository `pothos-dev/sapphire` exists on GitHub with the agreed visibility
- [ ] `origin` remote points at `pothos-dev/sapphire`
- [ ] The default branch is pushed and tracking upstream
- [ ] The pushed tree reflects the rename, theme, and README work (the app reads as "Sapphire" on GitHub, README renders with screenshots)
- [ ] User explicitly confirmed before the push
